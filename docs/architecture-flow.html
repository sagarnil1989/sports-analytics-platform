Timer Triggers (Azure Function App)

*/5 * * * * *
discover_cricket_inplay
Polls BetsAPI for live matches, updates FI list
→ bronze: active_inplay_fi/latest.json
*/5 * * * * *
capture_cricket_inplay_snapshot
Captures full snapshot for each live match — all odds, stats, lineups
→ bronze: inplay_snapshot/{eid}/fi={fi}/snapshot_{ts}.json + manifest.json
0 */1 * * * *
discover_cricket_upcoming
Fetches upcoming matches from BetsAPI, writes prematch index
→ gold: prematch/latest/index.json
10 */1 * * * *
capture_cricket_prematch_odds
Captures Bet365 prematch odds for each upcoming match
→ bronze: prematch_snapshot/{eid}/api_event_odds_summary.json
30 */5 * * * *
discover_cricket_ended
Checks innings_1.json staleness (>1 hr, not live) → auto-promotes to ended. No API call.
→ gold: innings_1_from_silver.json · ended/latest/index.json
*/10 * * * * *
parse_cricket_bronze_to_silver
Reads bronze manifests, parses raw JSON into structured silver files
→ silver: match_state · team_scores · market_odds · active_markets · innings_snapshot
*/10 * * * * *
build_cricket_gold_match_pages
Reads silver, builds match dashboard and innings tracker accumulator
→ gold: innings_1.json · match_dashboard.json · matches/latest/index.json
30 */1 * * * *
build_cricket_prematch_pages
Builds prematch gold pages from prematch bronze snapshots
→ gold: prematch/{eid}/match_dashboard.json · leagues/index.json

Column-Level Data Lineage

Gold / Silver Field Source File Origin API Field Description
match_state.time_status bronzeapi_event_view.json results[0].time_status "1"=live, "3"=ended. Gate for innings_1_from_silver.json write.
match_state.score_summary_events bronzeapi_event_view.json results[0].ss Final score string e.g. "165/10-169/3". Shown in ended view.
team_scores.runs / wickets bronzeapi_event_stats_trend.json stats[].score (format: "runs#wickets") Parsed by parse_runs_wickets(). Used in innings tracker chart.
market_odds.home_odds_decimal bronzeapi_event_odds.json odds[].home_od (market_key: "3_1") Match Winner 2-Way odds. Fallback source when active_markets has no odds.
active_markets.odds_decimal bronzeapi_live_market_odds.json results[].odds[].od (market_group_name contains "winner") Primary source for winner odds in innings tracker.
innings_tracker.home_team_odds silveractive_markets.json → market_odds.json (fallback) market_group_name contains "winner" OR market_key="3_1" Two-source resolution: tries active_markets first, falls back to market_odds.
innings_tracker.ball_window silverinnings_snapshot.json api_event_stats_trend.json last 6 deliveries Last 6 balls. Can be None — treated as empty list in view.
innings_tracker.over_from_score bronzeapi_event_odds.json odds[].ss (format: "143/2" → over derived from pg logic) Over number derived from odds score string via parse_over_from_pg().
ended_index.fi bronzeinplay_snapshot path: fi={fi} Directory name in blob path (no download needed) Bet365 FI ID. Extracted from bronze blob path via prefix scan.
ended_index.match_name goldinnings_1_from_silver.json tracker.match_name (set during silver parse) Required field — match skipped from ended index if missing.
match_dashboard.data_lineage bronzelineage.json (bronze) → lineage.json (silver) → gold API call timestamps, response sizes, bronze paths Full chain available at /api/matches/{event_id}/lineage
